# vitepress-template

## 介绍

[vitepress](https://vitepress.vuejs.org) 模板

## ✨ 特性

- 📌 开箱即用
- 💪 使用 Vue.js 最新特性开发
- 🐆 全面基于 Vite，速度够快
- 🖍️ markdown 中直接使用 vue 组件
- ✔️ 配置简单，上手容易
- 🚩 markdown 中直接使用 setup 语法糖
- 📃 第三方组件库支持

## 启动

```shell
pnpm i

pnpm dev
```
